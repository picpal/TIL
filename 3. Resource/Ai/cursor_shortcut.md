## 1. 공통/일반 단축키
기능(액션)	Mac	비고
커맨드 팔레트 열기	Cmd + Shift + P	VSCode와 동일
파일 열기 (Go to File)	Cmd + P	파일명 입력으로 빠른 검색/열기
사이드바 토글	Cmd + B	왼쪽 탐색기(Explorer) 토글
전체 검색(Find in Files)	Cmd + Shift + F	파일 전체 텍스트 검색
전체 치환(Replace in Files)	Cmd + Shift + H	검색 결과 치환
터미널 토글	Ctrl + ` (Mac/Win 동일)	VSCode와 유사 (Mac도 동일)
설정 열기	Cmd + ,	(Preferences)
바로가기 키 설정	Cmd + K → Cmd + S	키 바인딩 편집 창 열기
선 주석 토글 (라인 주석)	Cmd + /	블록 선택 상태에 따라 라인 단위 주석
현재 단어와 동일한 다음 단어 선택 (여러 번 누르면 여러 개 선택)  Ctrl + D  
커서를 괄호 끝으로 이동  Ctrl + Shift + \
괄호 안의 내용 선택  Shift + Alt + 오른쪽 방향키


## 2. 에디팅 관련 단축키
기능 (액션)	Mac 	비고
자동 완성 (Suggest)	Ctrl + Space	VSCode와 동일
코드 포맷팅 (Format)	Shift + Option + F	파일/선택 영역 포맷팅
파일에서 심볼 찾기	Cmd + Shift + O	함수/클래스 등으로 바로 이동
한 줄 이동	Option + ↑/↓	커서 위치 기준 줄 단위 이동
커서 여러 개	Option + 클릭	다중 커서 설정 (VSCode와 동일)
영역 접기/펼치기	Cmd + Shift + [ / ]	코드 블록 접고/펼치기
다음/이전 문제로 이동	F8 / Shift + F8	오류 / 경고 메시지 빠르게 확인
